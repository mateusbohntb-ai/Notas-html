# Notas-html


Vamos começar pela a estrutura de um simples html.
```<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>```

## <html></html>
O elemento <html></html>  é o elemento raiz da programação.

## <head></head>
esse elemento server para mostrar tudo que não e conteudo dentro da sua pagina.Incluindo ``palavras chave``  e css para dar estilo a sua pagina. 

##  <meta charset="UTF-8">
ele pode manipular qualquer conteudo textual que voce pode colocar.
